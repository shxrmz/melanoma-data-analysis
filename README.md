# Melanoma Risk Data Analysis

Statistical analysis of a real-world melanoma dataset completed as part of 
DATA1001 Foundations of Data Science at the University of Sydney (2025).

## What I did
- Data cleaning and exploratory visualisation in R
- Hypothesis testing: t-tests, chi-squared, ANOVA
- Produced a full HTML report using R Markdown

## Files
- .rmd — R Markdown source
- .html — Rendered report
